# OmniClip - 你的个人信息锚点

OmniClip 是一个专为**个人**打造的多平台内容聚合器。

在算法推荐泛滥、无限下滑、信息茧房无处不在的时代，OmniClip 是你的个人信息锚点。它剥离噪音、成瘾性 UI 和算法操控——你决定连接哪些平台，OmniClip 静默地收集、去重、用 AI 总结你真正关心的内容，汇聚成一份干净的、按时间排列的信息简报。

**不追热点，打破信息茧房。**

### 为什么叫 "OmniClip"？

- **Omni**：拉丁语"全部"。从碎片化的互联网各处聚合内容。
- **Clip**：如报纸剪报。裁掉噪音，只留高价值信息。

## 支持的平台

OmniClip 在所有平台上完全忽略算法推荐（如"为你推荐"、"发现"页面）。

| 平台 | 追踪内容 | 忽略内容 |
|------|---------|---------|
| **GitHub** | 你 **Star 的仓库**的最新 Release | Commits、Issues、算法推荐 |
| **YouTube** | 你**订阅频道**的完整视频 | `#shorts`、推荐视频 |
| **Twitter / X** | 你**关注的人**的时间线 | 算法注入内容、推广内容 |
